## 第一关：穿越雪地大森林 (The Frosty Forest Run)

### 核心玩法

- 圣诞老人从屏幕底部出发，游戏开始后**自动向前（向上）滚动推进**，玩家无法控制前进速度。
- 视角为**垂直俯视卷轴**，背景持续向下滚动模拟前进。

### 玩家操控

| 按键 | 动作 |
|------|------|
| ← / → (左右方向键) | 左右移动躲避攻击 |
| W | 发射法球攻击雪怪 |

### 敌人：雪怪 (Wildman)

- 从森林**左右两侧随机生成**，跑向中间雪道。
- 雪怪会向圣诞老人**投掷雪球（弹射物攻击）**。
- 雪怪可被玩家的法球击杀。

### 生命系统

- 圣诞老人共有 **10 点生命值 (HP)**。
- 每次被雪球击中 **扣 1 HP**（如有其他设定请修改此处）。
- HP 归零 → 触发**死亡特效** → 显示分数 → **Game Over**。

### 关卡结束条件

- **存活**：游戏进行一定时长后（具体时长待定，暂定 120 秒），前方出现城门。圣诞老人到达城门 → **过关，进入第二关**。
- **死亡**：HP 归零 → Game Over。

### 视觉与特效要求

- **雪花飘落粒子系统**（代码生成，不需要素材）：用 Pygame 绘制随机大小的白色圆点，赋予缓慢下落 + 横向摆动的运动轨迹，营造持续降雪氛围。
- 雪球飞行时带有**弹道轨迹特效**。
- 雪球命中时产生**撞击粒子效果**。
- 被击中时触发短暂**屏幕震动 (screen shake)**。
- 法球攻击需有发射与命中的视觉反馈。
- 背景环境：夜晚、雪地、密集松树林、冰冻地面。
- 随时间推移，敌人生成频率逐渐增加（难度递增）。

## 第二关：迷宫大逃亡 (Present Delivery Mission)

### 核心玩法

- 进入城门后，屏幕显示**三栋房子**供玩家选择。
- 每栋房子对应**不同的迷宫布局**（难度不同，选择带有运气成分）。
- 进入后，圣诞老人需在**规定时间内**穿过迷宫，找到**圣诞树**并放下礼物。

### 玩家操控

| 按键 | 动作 |
|------|------|
| ← / → / ↑ / ↓ (方向键) | 四方向迷宫移动 |

### 敌人：猎人 (Hunter)

- 迷宫中**随机生成**，会**主动追踪圣诞老人**。
- 猎人接触到圣诞老人 → **立即 Game Over**（一击必杀）。

### 计时系统

- 进入迷宫后启动**倒计时**（具体时长待定，暂定 180 秒）。
- 时间耗尽 → Game Over。

### 道具系统

- 迷宫中会**随机生成时间延长道具**。
- 拾取后**增加倒计时时间**（具体增加秒数待定，暂定 +15 秒）。

### 关卡结束条件

- **成功**：在时间内到达圣诞树并放下礼物 → 触发**庆祝特效** → 显示分数 → **游戏通关**。
- **失败 (被猎人抓住)**：立即 Game Over → 显示分数 → 重新从第一关开始。
- **失败 (时间耗尽)**：Game Over → 显示分数 → 重新从第一关开始。

### 视觉与特效要求

- 房子选择界面需有直观的视觉呈现。
- 迷宫内需有清晰的墙壁、通道视觉区分。
- 猎人追踪路径可带有视觉提示。
- 时间延长道具需有醒目的视觉效果（发光、闪烁等）。
- 通关时触发庆祝粒子效果（烟花、星星等）。
